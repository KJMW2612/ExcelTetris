# Excel Tetris Add-in (Excel T) 📊🕹️

Excel 2013~365용 현대식 테트리스 게임 및 비상 탈출용 추가 기능(VSTO Add-in)입니다. 
엑셀 전용 탭에서 버튼 클릭 한 번으로 게임판을 구성하며, 마우스 셀 선택을 원천 차단하고 왼손 조작 편의성과 보스 키(긴급 숨김) 기능을 탑재해 직장인의 오피스 환경에 자연스럽게 동화되도록 최적화되어 있습니다.

---

## 주요 기능 (Features)

* **긴급 숨김 모드 (보스 키)**: 게임 중이나 종료 상태일 때 `ESC`를 누르면 즉시 시트를 완벽하게 숨기고(`xlSheetHidden`) 원래 작업 중이던 업무용 시트로 화면을 강제 전환합니다.
* **흑백 필터 모드**: 컬러풀한 블록이 튀어 보이지 않게 단축키 `F`나 리본 메뉴를 통해 오피스 도표와 유사한 흑백 음영 톤으로 실시간 렌더링 필터를 씌워 위장합니다.
* **오피셜 가속 레벨링**: 누적 10줄을 제거할 때마다 레벨이 상승하며, 공식 테트리스 규격 드롭 지연 공식에 맞춰 물리적인 낙하 속도가 실시간으로 가속됩니다.
* **안전성 특화 제어 (STA 스레드 보호)**: 엑셀 셀 편집 중(텍스트 커서가 깜빡이는 상태) 시작을 누르더라도 크래시 없이 자동으로 편집 완료 처리 후 게임이 매끄럽게 시작됩니다.

---

## 설치 및 사용 환경 (Prerequisites)

* **운영체제**: Windows 10 / Windows 11 (C++ 메시지 훅 및 STA 스레드 연동 모델 특성상 Mac OS 미지원)
* **Excel 버전**: 데스크톱용 Excel 2013, 2016, 2019, 2021 및 Excel 365 (32비트/64비트 완전 호환)
* **필수 런타임**: .NET Framework 4.8 이상

## 📥 다운로드 및 설치 방법 (Installation)

> ⚠️ **중요 (필독)**: 인터넷에서 다운로드한 보안 제약으로 인해 아래 **차단 해제 단계를 반드시 수행**해야 정상 설치됩니다.

1. 본 페이지 우측의 **[Releases]** 탭에서 가장 최신 버전의 **`KJMW_ExcelTetris_v1.0.0.zip`** 파일을 다운로드합니다.
2. 다운로드한 ZIP 파일을 압축 해제하기 전, **파일을 마우스 우클릭 -> [속성]**으로 이동합니다.
3. 속성 창 맨 아래 보안 영역의 **`차단 해제 (Unblock)`** 체크박스를 체크한 후 **[확인]**을 누릅니다.
4. 압축을 해제한 후 폴더 내의 **`setup.exe`**를 실행합니다.
5. "게시자를 확인할 수 없습니다" 경고 창이 나타나면 **[설치]** 버튼을 눌러 설치를 완료해 주세요.

## 게임 방법 (How to Use)

1. 엑셀을 실행합니다.
2. 엑셀 상단 메뉴에 **[Excel T]** 탭이 생성되었는지 확인합니다.
3. 해당 탭 내에서 **[시작]** 단추를 누르면 자동으로 엑셀 화면이 최적 배율로 조절되며 테트리스 판이 구성되고 게임이 시작됩니다.
4. 아래 조작 가이드에 맞춰 왼손으로 플레이하며 즐깁니다:
   * **A / D**: 좌우 이동
   * **S**: 부드러운 하강 (소프트 드롭)
   * **Space**: 즉시 낙하 (하드 드롭)
   * **W / E**: 시계 / 반시계 회전
   * **Q**: 홀드 (블록 보관 및 교체)
   * **F**: 흑백 카무플라주 모드 온오프 (시트 기본 무지에 글씨 표시 전환)
   * **ESC**: 긴급 숨김 및 업무 복원 (보스 키)
